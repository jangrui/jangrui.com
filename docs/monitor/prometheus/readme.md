# Prometheus 是什么

Prometheus是一个最初在SoundCloud上构建的监控系统。自2012年成为社区开源项目，拥 有非常活跃的开发人员和用户社区。为强调开源及独立维护，Prometheus于2016年加入云原生云计算基金
会（CNCF)，成为继Kubernetes之后的第二个托管项目。
