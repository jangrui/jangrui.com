## 4.查看所有名称空间总的容器
::: details
```shell
kubectl get pods --all-namespaces -o jsonpath="{..image}" |\
tr -s '[[:space:]]' '\n' |\
sort |\
uniq -c
使用 tr 将空格替换为新的行
使用 sort 对结果排序
使用 uniq 对镜像使用计数
```
:::

## 6.查看命名空间
::: details
```shell
[root@master ~]# kubectl get namespace
# kubectl get ns 也可以
NAME              STATUS   AGE
default           Active   5d6h
kube-node-lease   Active   5d6h
kube-public       Active   5d6h
kube-system       Active   5d6h
ocp               Active   5d5h
```
:::

## 7.创建命名空间
::: details
```shell
[root@master ~]# kubectl create namespace summer
namespace/summer created
[root@master ~]# kubectl get namespace
NAME              STATUS   AGE
default           Active   5d6h
kube-node-lease   Active   5d6h
kube-public       Active   5d6h
kube-system       Active   5d6h
ocp               Active   5d5h
summer            Active   2s
```
:::

## 8.删除命名空间
::: details
```shell
[root@master ~]# kubectl delete namespace summer
namespace "summer" deleted
```
:::
