基于ansible全自动部署 redis5 三主三从集群，只需一条命令即可完成搭建，安装包可联系博主免费获取！
<!-- more -->

> 注意关闭防火墙（创建集群可能失败哟~）
## 执行
```shell
[root@summer work_yml]# ansible-playbook redis.yml
...
...
```
