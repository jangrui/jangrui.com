## cd / cd~ cd cd- 和cd..之间的区别
命令|  解释
-|-
cd     |   进入用户主目录 
cd ~   |  进入用户主目录 
cd -    |  返回进入此目录之前所在目录 
cd ..    | 返回上一级目录 
cd ../..  |返回上两级目录 
cd !$    |把上个命令的参数作为cd 参数使用 
cd /      |进入根目录
cd .      |当前目录
