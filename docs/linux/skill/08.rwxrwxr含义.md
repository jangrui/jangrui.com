- r:read 表示读权限 --数字4表示
- w:write 表示写权限 --数字2表示
- x:excute 表示执行权限 --数字1表示
第一个7：表示属主拥有4+2+1的权限也就是同时有读，写，执行权限
第二个7：表示与属主同一个组的所有用户拥有4+2+1的权限
第三个7：表示其他用户拥有4+2+1的权限
所以这里可以看出来这个777具有的威力（赋权一定要谨慎！）
再比如：rwxrwxr-- 774
