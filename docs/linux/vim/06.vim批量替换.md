## vim批量替换:

:s (substitute)命令用来查找和替换字符串。
语法如下：

>:{作用范围}s/{目标}/{替换}/{替换标志}

例如:%s/foo/bar/g会在全局范围(%)查找foo并替换为bar，所有出现都会被替换g
- 例子：
`:%s/source_pattern/target_pattern/g`
    把所有的summer换成love，那么:%s/summer/love/g就可以了。
