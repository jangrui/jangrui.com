# 18.linux命令notfound处理汇总

在实际操作linux系统特别是最小化安装或者是全新系统时，经常会碰到`-bash: ssh-copy-id: command not found`此类的报错。。。那么咱们应该安装哪些包才能让系统包含这些命令呢？ 

## ssh-copy-id

```text
yum install openssh-clients*
```

## ifconfig

```text
yum install net-tools
```

## make&&make install

```text
yum -y install gcc-c++
```
