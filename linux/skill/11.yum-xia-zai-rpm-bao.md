# 使用yum下载 RPM 包及其所有依赖

```text
# 获取vim的rpm包
yum install --downloaddir=/home/summer --downloadonly vim

# 安装
rpm -ivhU *  --nodeps --force命令即可实现服务安装

# 获取docker的rpm包
yum install --downloaddir=/home/docker/ --downloadonly docker-ce-18.06.0.ce
```

> --downloadonly只下载不安装 --downloaddir下载的rpm包的存放路径
